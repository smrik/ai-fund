# Weekly Loop Reviews

This folder holds Milestone 1 weekly-loop friction logs.

Convention:

- one file per counted or rehearsal session
- filename: `YYYY-MM-DD-session-NN.md`
- use `_template.md` for counted sessions
- guided CLI drafts may be named `YYYY-MM-DD-<TICKER>-friction-draft.md`

Rules:

- Every manual-data-surgery incident gets a fix or a ticket.
- The top friction item becomes the next week's fix candidate.
- Queue actions should be counted even when the PM rejects or defers everything.
- A session only counts toward M1 if it used real tickers and the PM completed the closeout notes.
