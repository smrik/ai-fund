# Execution Artifact Archive

This folder contains historical execution trackers, subplan briefs, and work logs.

It is not the canonical planning system anymore. The primary plan registry now lives in [docs/plans/index.md](../plans/index.md).

## Kept Here

- completed workstream trackers
- subplan briefs that supported a larger canonical plan
- old execution logs and handoff notes

## Completed Artifacts

Completed execution artifacts remain on disk under `docs/exec-plans/completed/`, but they are archival material rather than part of the published docs navigation.

Use the canonical plan registry in [docs/plans/index.md](../plans/index.md) first, and open specific execution artifacts from the repository only when you need historical implementation detail.
